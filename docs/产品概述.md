# 交换平台产品文档

## 用户故事

- 作为一个新玩家，我想**凑齐一套热门卡组**，以便在对战模式中拥有足够的强度
- 作为一个活跃玩家，我想**集齐图鉴/完成集卡任务**，以便拿到奖励

## 核心功能：

- 新手引导
- 标记自己需要/多余的卡牌
- 个人信息展示
- 游戏好友匹配
- 交换信息匹配
- 交换流程
